# KitchenMate

[KitchenMate](https://kitchenmate-efe45.web.app/) is a full-stack web application designed to manage recipes, compose complex dishes, and automatically generate structured shopping lists.
It was built primarily as a portfolio project and as a practical solution to a real-world problem: organizing personal recipes and multi-component meals in one system with automated ingredient aggregation.

While designed with food service professionals in mind, it is equally usable for home cooks who want structured meal planning and shopping automation.

## Features

- ✅ Full CRUD for Recipes, Dishes, and Shopping List items
- 📈 Recipe scaling (automatic ingredient recalculation)
- 🖼 Support for dish images
- ⭐ Favorite recipes and dishes
- 🛒 Generate shopping list from recipes or composed dishes
- 📄 Export recipes as PDF
- 📱 Fully responsive UI (mobile + desktop)
- 🧩 Custom modal system & animated UI interactions
- 🔐 JWT-based authentication (custom implementation)
- 💾 Persistent relational data storage
- 👤 User registration (create new accounts)
- 🔐 Password recovery with recovery code

## Tech Stack

### Frontend

- React (Vite)
- Tailwind CSS
- React Context and React Router
- Vitest

### Backend

- PHP (REST API)
- JWT Authentication & Authorization
- MariaDB

### DevOps

- Jenkins (Docker-based multibranch pipeline)
